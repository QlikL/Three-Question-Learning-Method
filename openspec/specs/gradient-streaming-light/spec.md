## ADDED Requirements

### Requirement: 全屏渐变流光背景层
系统 SHALL 在页面最底层渲染一个全屏固定定位的渐变流光背景层，使用多层 CSS 动画叠加实现柔和的浅色调渐变流动效果。背景层 MUST 设置 `pointer-events: none`，不拦截任何页面交互事件。

#### Scenario: 流光背景正常显示
- **WHEN** 用户打开页面且处于浅色模式
- **THEN** 页面背景显示柔和浅色渐变流光效果，包含多层动画叠加

#### Scenario: 流光背景不影响交互
- **WHEN** 流光背景正在播放动画
- **THEN** 用户可以正常点击页面上的所有按钮、链接和输入框，流光层不拦截任何事件

### Requirement: 多层动画效果
系统 SHALL 实现至少三种动画效果层叠加：
1. 大面积渐变旋转层（8s 周期）
2. 云朵光晕漂移层（12s 周期）
3. 光条扫过层（10s 周期）

#### Scenario: 渐变旋转层动画
- **WHEN** 页面加载完成
- **THEN** 第一层渐变以 `#fdfbfb → #ebedee → #f9f2e7 → #e0e7ff` 色系缓慢旋转，周期约 8 秒

#### Scenario: 云朵光晕效果
- **WHEN** 页面加载完成
- **THEN** 第二层显示柔和的浅粉/浅蓝光团，以不同速度缓慢漂移，形成云朵般的柔和光晕，周期约 12 秒

#### Scenario: 光条扫过效果
- **WHEN** 页面加载完成
- **THEN** 第三层显示明亮白色光线横向或斜向扫过页面，周期约 10 秒，形成光条扫过的视觉感

### Requirement: 颜色方案
系统 SHALL 使用柔和浅色调作为渐变色，主色系为 `#fdfbfb → #ebedee → #f9f2e7 → #e0e7ff`，光晕色为低饱和度的浅粉和浅蓝。

#### Scenario: 浅色模式颜色
- **WHEN** 用户处于浅色模式
- **THEN** 流光背景使用白、浅灰、淡黄、浅蓝紫等柔和色调，光晕透明度不超过 0.15

### Requirement: 主题切换适配
系统 SHALL 根据当前主题模式自动调整流光背景的显示状态。

#### Scenario: 浅色模式显示流光
- **WHEN** 用户切换到浅色模式
- **THEN** 流光背景层 `opacity` 为 1，动画正常播放

#### Scenario: 深色模式隐藏流光
- **WHEN** 用户切换到深色模式
- **THEN** 流光背景层 `opacity` 为 0，完全隐藏，深色背景保持一致

### Requirement: 动画性能
系统 SHALL 使用 CSS `transform` 和 `opacity` 动画，配合 `will-change` 属性触发 GPU 加速，确保动画流畅不卡顿。

#### Scenario: 60fps 流畅动画
- **WHEN** 流光背景正在播放动画
- **THEN** 页面主要内容区域渲染帧率不低于 60fps，无明显卡顿感
