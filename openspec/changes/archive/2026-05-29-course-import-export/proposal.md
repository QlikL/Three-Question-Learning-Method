# 课程导入导出功能

## 背景

用户需要将课程数据导出为文件备份，以及从文件导入课程数据。目前课程卡片的三个点菜单中"导出课程"功能尚未实现（显示toast提示），且"我的课程"页面缺少导入入口。

## 变更范围

### 1. 课程导出
- 在课程卡片三个点菜单中，点击"导出课程"将课程完整数据导出为 JSON 文件
- 导出内容包括：课程基本信息、知识图谱、心智模型、争议点、测评数据、知识库资料
- 文件名格式：`{课程标题}_{日期}.json`

### 2. 课程导入
- 在"我的课程"页面的排序方式右边添加"导入课程"按钮
- 点击按钮打开文件选择器，选择 JSON 文件
- 解析文件内容，验证数据格式有效性
- 将课程数据写入 IndexedDB，自动刷新课程列表
- 导入时检测课程ID冲突，生成新ID避免覆盖

## 不涉及的范围

- 不支持批量导入导出
- 不支持其他文件格式（仅JSON）
