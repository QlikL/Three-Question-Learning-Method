# 优化个人中心界面

## 背景

当前个人中心页面的三个核心模块存在逻辑和视觉问题：

1. **平均能力分计算逻辑有误**：当前是对所有课程（包括未完成的）的能力值求平均，应改为只统计已完成课程的课程评分平均分
2. **能力雷达图只有3个维度**（概念理解、批判思维、实践迁移），视觉简陋，需要扩展为6维度六边形雷达图
3. **学科柱状图按课程标题关键词匹配学科**，逻辑粗糙且分类不准确，应改为基于课程的学科属性进行分类
4. **课程数据模型缺少关键字段**：没有独立的课程评分字段、学科分类字段、以及课程体现的能力维度映射

## 变更范围

### 1. 课程数据模型扩展 (`Course`)
- 新增 `rating` 字段（0-100）：课程完成后的综合评分
- 新增 `subject` 字段：学科分类（如"计算机科学"、"数学"等）
- 将 `abilities` 从3维扩展为6维，分值范围从0-100改为0-10
- 新增 `abilityMapping` 字段：标记该课程体现哪几种能力

### 2. 课程生成流程更新 (`AiService`)
- 生成课程时同时生成学科分类和能力映射
- 课程完成时计算课程评分（基于答题表现）

### 3. 能力雷达图重写 (`RadarChart`)
- 扩展为6维度六边形
- 分值范围0-10，数据来源为已完成课程的平均值
- 美化视觉效果（渐变填充、动画、更好的配色）

### 4. 学科柱状图重写 (`BarChart`)
- 按学科字段分组（而非标题关键词匹配）
- 分值为该学科已完成课程的评分平均分（0-100）
- 美化视觉效果

### 5. 平均能力分逻辑修正 (`ProfilePage`)
- 只统计已完成课程
- 平均能力分 = 已完成课程的 rating 平均值

## 不涉及的范围

- 课程学习流程（三问）不变
- 课程列表页面不变
- AI聊天组件不变
