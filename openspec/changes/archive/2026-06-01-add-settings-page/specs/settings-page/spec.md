## ADDED Requirements

### Requirement: 设置入口按钮显示
系统 SHALL 在个人中心页面标题右边显示设置齿轮图标按钮（⚙）。

#### Scenario: 设置按钮正确显示
- **WHEN** 用户进入个人中心页面
- **THEN** 在页面标题右边应显示设置齿轮图标按钮（⚙）

#### Scenario: 点击设置按钮跳转
- **WHEN** 用户点击设置按钮
- **THEN** 系统应跳转到设置页面

### Requirement: AI 模型配置
系统 SHALL 提供 AI 模型配置功能，支持设置 API Key 和选择模型。

#### Scenario: 显示当前配置
- **WHEN** 用户进入设置页面的 AI 配置模块
- **THEN** 系统应显示当前配置的 API Key（隐藏部分字符）和模型名称

#### Scenario: 修改 API Key
- **WHEN** 用户输入新的 API Key 并点击保存
- **THEN** 系统应保存新的 API Key 并显示成功提示

#### Scenario: 选择 AI 模型
- **WHEN** 用户从下拉列表中选择一个 AI 模型
- **THEN** 系统应保存选择的模型并显示成功提示

#### Scenario: 配置验证
- **WHEN** 用户输入无效的 API Key（如格式错误）
- **THEN** 系统应显示错误提示，不保存配置

### Requirement: 主题切换
系统 SHALL 提供深色模式和浅色模式切换功能。

#### Scenario: 显示当前主题
- **WHEN** 用户进入设置页面的主题模块
- **THEN** 系统应显示当前使用的主题（深色/浅色）

#### Scenario: 切换到深色模式
- **WHEN** 用户选择深色模式
- **THEN** 系统应切换到深色主题并保存设置

#### Scenario: 切换到浅色模式
- **WHEN** 用户选择浅色模式
- **THEN** 系统应切换到浅色主题并保存设置

#### Scenario: 主题持久化
- **WHEN** 用户刷新页面
- **THEN** 系统应保持用户选择的主题设置

### Requirement: 学习偏好设置
系统 SHALL 提供测验难度偏好设置功能。

#### Scenario: 显示当前难度
- **WHEN** 用户进入设置页面的学习偏好模块
- **THEN** 系统应显示当前的测验难度偏好

#### Scenario: 设置测验难度
- **WHEN** 用户选择一个测验难度（简单/中等/困难）
- **THEN** 系统应保存难度设置并显示成功提示

### Requirement: 数据管理
系统 SHALL 提供数据管理功能，包括清除数据、导出和导入。

#### Scenario: 清除学习数据
- **WHEN** 用户点击"清除学习数据"按钮并确认
- **THEN** 系统应清除所有学习数据并显示成功提示

#### Scenario: 导出所有课程
- **WHEN** 用户点击"导出所有课程"按钮
- **THEN** 系统应将所有课程数据导出为 JSON 文件并下载

#### Scenario: 导入课程数据
- **WHEN** 用户点击"导入课程"按钮并选择 JSON 文件
- **THEN** 系统应导入课程数据并显示成功提示

#### Scenario: 清除数据确认
- **WHEN** 用户点击"清除学习数据"按钮
- **THEN** 系统应显示确认对话框，提示数据不可恢复

### Requirement: 关于与帮助
系统 SHALL 提供关于与帮助信息。

#### Scenario: 显示版本信息
- **WHEN** 用户进入设置页面的关于模块
- **THEN** 系统应显示当前版本号和应用名称

#### Scenario: 显示使用帮助
- **WHEN** 用户进入设置页面的帮助模块
- **THEN** 系统应显示简要的使用帮助信息

#### Scenario: 显示反馈入口
- **WHEN** 用户进入设置页面的反馈模块
- **THEN** 系统应显示反馈链接或联系方式
