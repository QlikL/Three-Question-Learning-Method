# 设计方案：完善项目文档

## 1. README.md 内容规划

### 1.1 项目简介
- 项目名称：三问高效学习法
- 一句话描述：以用户提问为核心触发机制的智能学习工具
- 核心理念：三问法（心智模型→学术分歧→深度测评）

### 1.2 功能特性
- 智能课程生成（AI驱动）
- 知识图谱可视化
- 根本性分歧探索
- 深度理解测评（六维能力体系）
- 知识库管理（AI总结 + 用户上传）
- 课程导入导出
- AI助手聊天
- 个人中心（雷达图+柱状图）

### 1.3 快速开始
- 克隆/下载项目
- 配置AI API Key（参考 AI配置指南.md）
- 启动方式（直接打开index.html或使用Live Server）

### 1.4 项目结构
- 目录结构说明
- 核心文件说明

### 1.5 技术栈
- Vanilla JS + HTML5 + CSS3
- Canvas 2D（图谱/图表渲染）
- IndexedDB + localStorage
- 阿里云百炼 API
