# 阿里云百炼 AI 配置指南

## 📋 配置步骤

### 1️⃣ 获取 API Key

1. 访问阿里云百炼平台：https://bailian.console.aliyun.com/
2. 登录你的阿里云账号
3. 进入"API-KEY管理"页面
4. 点击"创建API-KEY"
5. 复制生成的 API Key

### 2️ 填写配置信息

打开文件：`js/services/aiService.js`

找到第 23-27 行的配置区域：

```javascript
// ==================== 配置区域 ====================
// ⚠️ 请在这里填写你的阿里云百炼 API Key
API_KEY: 'YOUR_API_KEY_HERE',

// ⚠️ 请在这里填写你要使用的模型名称
MODEL_NAME: 'qwen3-8b',  // 推荐：qwen3-8b（均衡）或 qwen-plus（高质量）
// ==================================================
```

**将 `YOUR_API_KEY_HERE` 替换为你的实际 API Key**

例如：
```javascript
API_KEY: 'sk-1234567890abcdef1234567890abcdef',
```

### 3️⃣ 选择模型

阿里云百炼提供多个通义千问模型：

| 模型名称 | 特点 | 适用场景 | 推荐指数 |
|---------|------|---------|---------|
| `qwen3-8b` | 速度快，性价比高 | **大多数场景（推荐）** | ⭐⭐⭐⭐⭐ |
| `qwen-plus` | 效果好，速度均衡 | 高质量要求 | ⭐⭐⭐⭐ |
| `qwen-turbo` | 速度最快，成本最低 | 简单任务、快速响应 | ⭐⭐⭐⭐ |
| `qwen-max` | 效果最好，成本最高 | 复杂推理 | ⭐⭐⭐⭐ |

**建议：默认使用 `qwen3-8b`，性价比最优**

### 4️⃣ 测试配置

1. 刷新网页
2. 在首页搜索框输入一个问题，例如："如何学习JavaScript"
3. 观察：
   - ✅ 如果成功，会显示AI生成的课程信息
   - ❌ 如果失败，浏览器控制台会显示错误信息

## 🔍 常见问题

### Q1: 如何确认AI是否启用？

打开浏览器控制台（F12），如果看到：
- "AI服务调用失败，降级到模拟数据" → 说明AI未配置或配置错误
- 没有这个提示 → AI正常工作

### Q2: API调用失败怎么办？

检查以下几点：
1. API Key是否正确（不要有空格）
2. 模型名称是否正确
3. 网络连接是否正常
4. 查看浏览器控制台的详细错误信息

### Q3: 费用问题

阿里云百炼按调用量计费：
- qwen-turbo：约 0.008元/千token
- qwen-plus：约 0.04元/千token
- qwen-max：约 0.12元/千token

学习项目使用量不大，费用很低。

### Q4: 如何切换回模拟数据？

只需将 `API_KEY` 改回 `'YOUR_API_KEY_HERE'` 即可。

## 🎯 实现的功能

当前AI服务支持以下功能：

1. ✅ **生成课程信息** - 根据用户问题自动生成课程标题、学科分类和六维能力评估
2. ✅ **生成知识图谱** - AI生成知识点和关系网络（支持分层辐射式布局）
3. ✅ **提取心智模型** - 提取学科核心心智模型和底层逻辑框架
4. ✅ **挖掘学术争议** - 发现学术界的争议点、正反方观点及支撑论据
5. ✅ **生成测评题** - 基于布鲁姆认知层级自动出题和答案解析
6. ✅ **知识库AI总结** - 上传资料后AI自动生成结构化总结文档
7. ✅ **AI助手聊天** - 全局AI对话面板，支持上下文关联问答
8. ✅ **节点详细信息** - 知识图谱节点点击后可调用AI生成详细解释
9. ✅ **AI解析错题** - 测评答错后AI给出详细解析和正确答案说明

所有功能都支持**自动降级**：AI调用失败时自动使用模拟数据，不影响使用。
