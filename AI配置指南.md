### 1️⃣ 获取 API Key

根据你选择的服务商，获取对应的 API Key：

#### 阿里云百炼
1. 访问：https://bailian.console.aliyun.com/
2. 登录后进入"API-KEY管理"
3. 创建并复制 API Key

#### 百度文心一言
1. 访问：https://console.bce.baidu.com/qianfan/
2. 创建应用获取 API Key 和 Secret Key

#### 讯飞星火
1. 访问：https://console.xfyun.cn/
2. 创建应用获取 API Key 和 App ID

#### 智谱 AI
1. 访问：https://open.bigmodel.cn/
2. 注册后获取 API Key

#### 腾讯混元
1. 访问：https://console.cloud.tencent.com/hunyuan/
2. 开通服务获取 API Key

#### 月之暗面 Kimi
1. 访问：https://platform.moonshot.cn/
2. 注册后获取 API Key

#### DeepSeek
1. 访问：https://platform.deepseek.com/
2. 注册后获取 API Key

---

### 2️⃣ 在设置页面配置

1. 打开项目首页
2. 点击右上角头像进入**个人中心**
3. 点击**设置（⚙）**图标
4. 在**AI 模型配置**区域：
   - 选择服务商
   - 输入 API Key
   - 选择模型（或手动输入模型名称）
   - 点击「保存配置」

---

### 3️⃣ 测试配置

**方式一：使用数据管理工具（推荐）**

1. 打开 `clear-cache.html` 页面
2. 点击「测试AI连接」按钮
3. 观察：
   - ✅ 如果成功，会显示"AI连接成功"及AI回复内容
   - ❌ 如果失败，会显示具体错误信息

**方式二：在首页测试**

1. 返回首页
2. 在搜索框输入问题，例如："如何学习JavaScript"
3. 观察：
   - ✅ 如果成功，会显示 AI 生成的课程信息
   - ❌ 如果失败，会提示"AI服务调用失败，降级到模拟数据"

## 💾 保存多个配置

你可以保存多个 AI 模型配置，方便快速切换：

1. 配置好一个服务商后，点击「保存当前配置」
2. 输入配置名称（如"阿里云 qwen-plus"）
3. 保存后会在下方显示已保存的配置列表
4. 点击「切换」即可快速切换

## 🔍 常见问题

### Q1: 如何确认 AI 是否启用？

- 首次打开时，如果未配置 AI，会弹出提示"首次使用先配置AI模型哦"
- AI 调用失败时，会显示"AI服务调用失败，降级到模拟数据"
- 正常情况下，AI 生成的课程信息会显示在课程卡片中

### Q2: API 调用失败怎么办？

检查以下几点：
1. API Key 是否正确（不要有多余空格）
2. 模型名称是否正确
3. 网络连接是否正常
4. 查看浏览器控制台（F12）的详细错误信息

### Q3: 费用问题？

各服务商按调用量计费，学习项目使用量不大，费用很低。建议使用免费额度或低成本模型。

### Q4: 如何切换回模拟数据？

只需清空设置页面中的 API Key 即可，项目会自动降级到模拟数据模式。

### Q5: 自定义服务商如何配置？

1. 选择"自定义（OpenAI兼容）"
2. 输入 API 端点（如 `https://api.example.com/v1/chat/completions`）
3. 输入 API Key
4. 输入模型名称（如 `gpt-4`）

## 🎯 实现的功能

当前 AI 服务支持以下功能：

1. ✅ **生成课程信息** - 根据用户问题自动生成课程标题、学科分类和六维能力评估
2. ✅ **生成知识图谱** - AI 生成知识点和关系网络（支持分层辐射式布局）
3. ✅ **提取心智模型** - 提取学科核心心智模型和底层逻辑框架
4. ✅ **挖掘学术争议** - 发现学术界的争议点、正反方观点及支撑论据
5. ✅ **生成测评题** - 基于布鲁姆认知层级自动出题和答案解析
6. ✅ **知识库 AI 总结** - 上传资料后 AI 自动生成结构化总结文档
7. ✅ **AI 助手聊天** - 全局 AI 对话面板，支持上下文关联问答
8. ✅ **节点详细信息** - 知识图谱节点点击后可调用 AI 生成详细解释
9. ✅ **AI 解析错题** - 测评答错后 AI 给出详细解析和正确答案说明
10. ✅ **课程学习评估** - 基于五维度学习数据生成 AI 评估报告

所有功能都支持**自动降级**：AI 调用失败时自动使用模拟数据，不影响使用。
